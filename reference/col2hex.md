# Convert colour to hex value

go from a colour like "red" to the hex code, "#FF0000"

## Usage

``` r
col2hex(colour)
```

## Arguments

- colour:

  a colour name from
  [`colours()`](https://rdrr.io/r/grDevices/colors.html) or
  [`colors()`](https://rdrr.io/r/grDevices/colors.html)

## Value

A hex color hex value

## Examples

``` r
col2hex("red")
#> [1] "#FF0000"
```
